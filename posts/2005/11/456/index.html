Zoundry升级了

2005年11月03日 · 1 分钟阅读 · 490 字

今天打开Zoundry的时候，发现它已经升级到0.9.137了，赶紧下一个来用。

官方资料里面特别提到了段落操作的改进：

New Enter key behaviour:
Pressing Enter key starts a new paragraph (

); Pressing Shift+Enter inserts a line break (
) within the same paragraph. Use Shift+Enter if you want to start a new line on the same paragraph.

按回车键开始一个新的段落（

）；按Shift+回车键是在同一个段落里面换行（
）

嘿，两周前我还给他们写过信，说这个事情，他们回信说1、2个月内解决，他们速度还是挺快的，现在这个操作方式方便多了，也更像Dreamweaver了。

另外，Zoundry对自己的不足也作了一下改进：

  • 比如自己不支持复杂HTML（比如table）的编写，就提供了OpenHtmlFile的方法，让用户可以直接打开编辑好的HTML
  • 增加了PasteAsHtml，让用户可以选择直接粘贴HTML内容，或仅粘贴文本内容
  • 为图片加链接的时候不需要鼠标拖拽成高亮状态了，直接在图片上右键点"Create Link"就行了

以上是我比较感兴趣的更新，其他更新请看这里：http://www.zoundry.com/releasenotes.html

不知道Zoundry以后能不能支持多语言包，这样就可以彻底推广了。
