Zoundry新版本支持MSN Space

apiIMTagTag

昨天刚在《国内的BSP还在等什么？》写过，Zoundry要发布新版软件支持MSN Spaces的API，没想到今天新版就出来了，看他的更新说明：

This release contains some minor enhancements as well as bug fixes.

  1. Added official support for MSN Spaces. To use MSN Spaces, choose MSN Spaces for account type.
  2. Fixed create link to existing post bug where the currently selected text is over written with the post title.

用了一下，并不是想象中那样的更新，Zoundry只在新建Blog帐号的支持系统项目里增加了MSN Spaces，且不需要像Dareo在另外两个Blog客户端里演示使用那么麻烦，都不用填写XMLRPC地址、SSL和端口。

Zoundry的反应速度真是挺快的，不过感觉这个软件的发展方向并不是像Blogjet那种开放型的，是一个相对封闭的，主要是因为没有开放SSL、端口等高级选项，但是对现有特定服务的支持却很好，尤其他这种对高级选项的封装对最终用户有很好的可操作性。

OK，以后又可以多一个Blog可以通过Zoundry更新了，哪个BSP是下一个？
