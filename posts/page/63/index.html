开始试用Firefox 2.0

近两天比较郁闷，因为我的Firefox 1.5不能正常访问Gmail了，而且没听说别人出这个问题。我可是一直用Firefox访问Gmail的，这件事一定要搞定~

今天Gtalk上问Keso大哥能不能正常访问Gmail，他说能，忙问版本，答曰：2.0，恍然大悟，马上去下了一个Firefox 2.0b1的中文版，装上后马上连Gmail，结果正常。

之后下了Firefox Portable 2.0 Beta 1，是个英文预览版，凑合先用了，因为没在Mozilla找到中文版的zip包，就用之前安装 的2.0b1文件替换了Portable自带的英文版，好了，中文版的Firefox 2.0 Portable搞定了。

2.0b1的支持的扩展还少得可怜，强行装了几个低版本的扩展，都以失败告终，正常启用的扩展（以后随时增加）：

  • 365key 天天网摘右键收藏 0.2.1 这可是2.0专用的
  • Adblock Plus 0.7.1.1
  • CustomizeGoogle 0.49
  • DOM Inspector 1.8.1b1
  • FlashGot 0.5.9.995
  • Gmail Notifier 0.5.6.3 有了它我就安心多了
  • MR Tech Local Install 5.2 这个还是一如以往的好用
  • NoScript 1.1.4.1
  • Organize Status Bar 0.3
  • Pearl Crescent Page Saver Basic 1.2
  • Searchbar Autosizer 1.1 兼容，但是没见起作用
  • SearchStatus 1.16
  • Tab Mix Plus 0.3.0.60805 是测试版，支持2.0，来自Tab Mix的论坛

毕竟这个版的Firefox还是测试性质的，有很多地方不太令人满意：

  1. 兼容扩展太少，如上，不说了
  2. 发现Firefox 2.0b1这个版对Ajax的支持还不好，像我的Wordpress后台特效全没了，隐藏的菜单都显示出来了，看来除了之前出来的IE 7.0要适配，连Firefox 2.0也要配一下了~
  3. Feed全被Firefox接管了，只要是Feed，一律都由Firefox的内置方案渲染，根本不管是不是经过Feedburner、Feedsky加工过的
  4. 搜索栏感觉设计得不好，不如1.5的人性化

Update：多亏Keso大哥提醒，借助MR Tech Local Install 5.2，很多扩展都装上了，乌拉~

求助：我的Firefox无法正常访问Gmail了

昨天早上开始，我工作用的Firefox 1.5.0.6访问Ajax版Gmail出错，提示信息如下：

Loading…

This seems to be taking longer than usual.

If you are using a slow Internet connection, you can wait a bit longer for this page to finish loading, or just use basic HTML view for now.

If you are using your normal Internet connection and you usually get past this loading step without any problems, please refresh this page in your browser. If you continue to have trouble loading your account, please visit the help center for troubleshooting information.

跟着里面的链接，点到了Gmail的帮助信息，看了一下上面列的可能出现冲突的第三方软件，除了Maxthon（Maxthon访问反而正常？！）我一个都没装，根据提示这种情况可能是缓存闹得，就按提示清空Firefox的缓存（看了才知道我根本就没使用缓存，那里设置的缓存容量是0M）。

按照提示信息" 某些情况下，可能需要多次清空您的缓存，才能彻底解决这一问题"，我登出又登入了好多次，缓存也清空了无数次，而且http://mail.google.com/和https://mail.google.com/都试过了，还是不行。

昨天回家发现家里的Firefox也不能访问Gmail，真是奇怪了。

所以只好在这里求救，看看哪位达人可以拯救我，知道的请在这里留言，或者使用Gtalk联系我（94smart）~

Update：还是不知道哪里出的问题，FireBug查出Gmail的页面有一个js错误，好像是导致Gmail Loading剧卡的关键~

Update2：刚才试了一下2.0b1，居然可以用，近期准备全部切换至Firefox 2.0

Update3：今天（8-17）偶然试用Firefox 1.5访问Gmail，居然没问题了，真是恢复得莫名其妙~
