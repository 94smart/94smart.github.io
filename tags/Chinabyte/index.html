2007-10-24 闲扯Wordpress.com.cn的来历

Blog Blog Chinabyte Wordpress Wordpress Wordpress.com.cn 天极网 随笔

今天早上在Gmail收信，收到一封标题为wp-testers-request的英文信，说的是邀请加入Wordpress.com.cn开Blog，还以为是垃圾信，看发信人信箱居然是mail@chinabyte.com，这时我才恍然大悟，原来Wordpress.com.cn是天极网搞的呀~ 一年多前就出来的这个Wordpress.com.cn，当时好多人以为是Donews做的，没想到是天极的。但如果这个BSP已经一年多了，为什么信件内容还是Wordpress自带的英文内容，没人汉化吗？ 真希望不要白白浪费这么一个品牌域名呀~
