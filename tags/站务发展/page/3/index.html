我也想做电子杂志

前一阵还看了herock的Web 2.0周刊和Sohu的Blog周刊，就有一种感觉，现在做这样的东西可能比较合适。 想做电子杂志（或电子报、线上杂志，随便怎么叫吧），是上周看物志的时候产生比较强烈的想法。 唉，老毛病又犯了，见到什么好，也想自己有一个。 今天又看到liuren要出版刘韧电子报，终于坐不住了，很想去做一个用来自娱自乐。 liuren的想法是：（节选自上述Blog）

电子报是你用RSS办的一份媒体。你是主编，办报的成本=你挑选几个Blog的RSS就行。 比如，我会将洪波、王建硕、老冒、蒋涛、粱宁、韩磊、登高、陈年、学凌，当作我的作者，将他们的RSS聚合到我Blog左栏。

我的想法是，使用我365key上的收藏，其实每天都会做“昨日收集”这样的网摘汇总，如果变一下形式，是不是可以做成一个小型的电子杂志？这样的话，我可以出个日报日刊什么的，比周刊有意思吧？嘿嘿~ 顺便自嘲一下：再次，做完了也还有我自己看。
