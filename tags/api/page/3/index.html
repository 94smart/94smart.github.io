有些事还就Google干得出来

我指的是Google的Passport API，全称Google Account Authentication，分为客户端和网页身份验证。不管Google什么企图，还是挺让人佩服的。

去年写过一篇跟Passport有关的文章，当时是挺想做一个Passport系统，把其它网站的用户接进来，然后再共享出去。

当时技术实现都想好了，也拉了人想去做，后来突然发现有一个重大问题没有考虑清楚，就是用户从哪来的问题，除非你有大量用户跟使用该Passport系统的网站交换，否则一切都是胡扯，别的网站不会轻易和你共享用户，毕竟做网站的对用户资料都是看得挺重的。

这种设想肯定有不少人都作过，但是都在很多环节上放弃了，因为这件事并不是谁都可以做的，之前微软和SUN作过类似的尝试，但是感觉还是在自己的服务上用得不错，没法给其它网站使用。

而现在，Google正在做这个事情，我觉得成功的机率很高，因为Google有别人提供不了的如此众多的用户资料，以及不做恶的口碑，使其它网站都会愿意与Google共享。虽然现在实现的跟微软的Passport差不太多，相信随着Google开发更多的资源、优化各种流程后，Google的Passport会有一个不错的发展。

此举应该也会刺激到微软、Yahoo及其他网络巨头，使他们继续优化各自的服务，增强他们的Passport系统，搞不好未来只有那么几种公用的Passport系统：Google、微软Live、Yahoo……

今天的YY到此结束~

p.s.Zoundry最近疯狂的更新，又预示着什么呢？

Yahoo My Web2.0 beta和Del.icio.us小议

Yahoo收购Del.icio.us和Flickr也有一段时间了，并没有让人看到稍大规模的整合，不过前天出来的新版My Web2.0倒是用到了Del.icio.us，让我们看到了些些整合的可能。

更正：上面是我的错觉，之前没怎么用过My Web2.0，我以为近期才推出导入功能，不过不太影响后面的内容，下面就不改了。感谢xobo提醒！

在Tools里面看到My Web2.0支持Del.icio.us的内容导入，我马上点了进去，然后看到一个帮助信息：

  1. 去http://del.icio.us/api/posts/all
  2. 将页面使用"Save as"另存在本地
  3. 定位本机存储的文件地址
  4. 选择共享的范围：
    仅自己
    家庭
    朋友
    任何人
  5. 点击导入按钮

挺奇怪的，这种导入操作难道不能直接由系统进行吗？非得自己下载再上传吗？很有可能是初步整合失败的产物。

我猜这只是整合的前奏，后面的可能会发展为My Web+Del.icio.us+Flickr+360的格局。

Zoundry新版进化

几天没在，Zoundry就出了个新版，这版跨过了1.0，直接出了个1.0.23，修正了一批Bug，还推出了一些新功能，看来Zoundry还想将Beta继续下去，我原以为过了1.0会把Beta撤下来~

看看都有什么改进：

  1. 拼写检查（词典要自己下载）
  2. XHTML源码编辑及自动完成
  3. XHTML代码规范验证工具
  4. XHTML整理操作
  5. 支持语言本地化
  6. 查找/替换
  7. 提供MovableType API和WordPress文本实体扩展（例如摘要操作）
  8. 增加对Blog实体的排序方法（如按时间、按字母顺序等）
  9. 建立Blog帐号的时候增加支持RSD来识别API设置
  10. 其它修改，包括图片快速对齐菜单，用"Link to Posts"增加旧帖子的链接等

原文：http://www.zoundry.com/releasenotes.html

这个新版改动还是挺大的，主要是增加了源码编辑操作，又向Dreamweaver迈进了一步~界面改动不太大，用起来还是挺习惯的，希望下次改进不要再改动编辑区了，越改越复杂。

最令我感觉人性化的操作是崩溃恢复了，刚才写这篇的时候使用了XHTML代码验证，还使用的是严格式（Strict）的，一点就报错还自己关闭了，然后我就傻眼了，写的内容都没保存，结果再次打开Zoundry，发现内容还好好的在编辑区里，真是高兴死了。

还有就是本地化方面，Zoundry并没有采取其它软件的本地化策略，没有提供语言包，而是提供了开放型的解决方法，让用户去自己解决，想改什么就改什么。不过我觉得这种方法有点取巧，虽然很快就会有用户自己定制的语言包出来，但是却让大部分用户决得不够方便。

总的来讲，Zoundry在进化，但是方向并不是一直朝Blog编辑器去的，而是向专业网页编辑器（Dreamweaver），或者更夸张点说是向专业文本编辑器（Word？）去的，越做越高端，不清楚是不是要走高端路线了。

不过话说回来，这个编辑器还是值得继续推荐的，快下手~
