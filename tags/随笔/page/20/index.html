小试Prototype：为静态文章列表加上标志

昨天，用Prototype.js解决了一个以前一直解决不了的问题，即静态文章列表加上New标志。

工作上一直使用静态的CMS（内容发布系统），好处是发布的内容全部都有静态页面、唯一地址，访问速度也比较稳定，缺点是放弃了动态的效果，比如文章动态链接、列表更新等等，以及今天要说的根据文章发布时间给标题加New标志。

原理是用Javascript分析列表代码里面带有时间的条目，时间是不是在时间限制之后，如果是，就在这个条目的时间代码段前面加上New的标志代码。

有些事还就Google干得出来

我指的是Google的Passport API，全称Google Account Authentication，分为客户端和网页身份验证。不管Google什么企图，还是挺让人佩服的。

去年写过一篇跟Passport有关的文章，当时是挺想做一个Passport系统，把其它网站的用户接进来，然后再共享出去。

当时技术实现都想好了，也拉了人想去做，后来突然发现有一个重大问题没有考虑清楚，就是用户从哪来的问题，除非你有大量用户跟使用该Passport系统的网站交换，否则一切都是胡扯，别的网站不会轻易和你共享用户，毕竟做网站的对用户资料都是看得挺重的。

这种设想肯定有不少人都作过，但是都在很多环节上放弃了，因为这件事并不是谁都可以做的，之前微软和SUN作过类似的尝试，但是感觉还是在自己的服务上用得不错，没法给其它网站使用。

而现在，Google正在做这个事情，我觉得成功的机率很高，因为Google有别人提供不了的如此众多的用户资料，以及不做恶的口碑，使其它网站都会愿意与Google共享。虽然现在实现的跟微软的Passport差不太多，相信随着Google开发更多的资源、优化各种流程后，Google的Passport会有一个不错的发展。

此举应该也会刺激到微软、Yahoo及其他网络巨头，使他们继续优化各自的服务，增强他们的Passport系统，搞不好未来只有那么几种公用的Passport系统：Google、微软Live、Yahoo……

Opera 9，可以用用

Opera 9发布几天了，下了一个试了试，还不错，感觉Opera在用户体验方面做得越来越好。

Opera 9的一大亮点是支持组件（Widgets），初看上去是学FireFox的扩展，其实不只这样，Opera还参考了Yahoo和微软的桌面组件，将很多娱乐功能搬到了浏览器里。

与FireFox的扩展相比，Opera的组件并不集成在浏览器上，而是单独开一个小窗口，可以与浏览器分开显示。昨天发现一个不错的组件：Pandora Radio Widget，有了它，就可以将Pandora的大窗口缩成一个小条的播放器面板，爽~

我还关注的一个功能是BT集成，宣传文档里面说从此不再需要其他的BT客户端软件了，用Opera就可以搞定，我觉得Opera是浏览器产品里比较有前瞻性的，现在BitTorrent都与正版发行商合作了，以后的发展应该更好，这样做不管从哪个角度说都是正确的，不过让我想起了微软和IE的做法……

Opera 9还有很多新增功能，在此就不赘述了，不管喜欢IE还是喜欢FireFox的都可以去试试。

最后扯两句，Opera自从宣布免费以后，在社区应用方面的发展很迅速，走了一条与IE、Firefox都不大相同的浏览器之路，即软件、服务、社区的集成，也许这种模式只有IE的后续版本依靠live.com可以去打败。扯完~

再说E准，想到哪儿说到哪儿

之前小小推荐了一下E准，最近又用了用，想写一点补充的想法。

首先，我感觉这次测试版的推广效果还不错，正好赶上世界杯这个热潮，时间刚刚好，有那种顺水推舟的感觉。

其次，前几天看到詹老师写E准电视社区的文章，社区是发展趋势，我完全同意。但是，我不认为WAP和3G应用会是E准的主要方向，觉得E准瞄准的方向应该是电视、IPTV，或者说最起码应该是机顶盒、视频点播类的应用。

试想一下，如果E准可以增加（或合作伙伴提供）节目的点播功能，在E准电视社区的基础上，结合用户推荐等机制，其他用户不止会对推荐节目产生好印象，还可以直接浏览，从而增加该节目的收视率。

另外，在视频点播的基础上，利用现有的"我的日历"还可以提供播放日程，自己定制节目表，到时间就自动播放相应节目，多爽~

还有，如果我现在是电视台的，我就需要考虑了，是不是找E准来外包节目播出表相关的服务，包括节目评价、推荐等等一套互动功能，还是决定自己赶紧做一个电视社区，晚了就没机会了。

最后，给E准提点使用上的建议：

  1. 赶紧支持Firefox吧~
  2. 在我的频道里面默认显示有节目的节目单
  3. 节目搜索结果的选择框默认最好清空，如果可以实现点击条目就选择，还支持多选、组合键（CTRL、SHIFT等）选取就更好了。
  4. 搜索结果选中后可以批量设置提醒或加入节目单

上面的思维顺序有点混乱，凑合看吧，还有我不是很了解IPTV的定义、术语，如果说错了请各位指正。
