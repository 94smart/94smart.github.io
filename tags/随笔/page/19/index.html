参加微软的网络阅读行为访谈

被拍了，再前一个星期六的上午，在朝外的良木缘咖啡馆，当时我正通过免费的无线网连接Bloglines，在前面有个数码录音，后面有个DC摄像，不断记录我的各种操作及操作的原因、想法。

这是我的又一次线上转线下的活动，微软研究院的吕昊通过Gmail找到我，希望我能参加他们的RSS阅读访谈，我们在Gtalk上沟通，最后在Google Calendar上安排好了时间和地点，有一点很奇怪，为什么不用MSN呢？

起先并不想去，但是他们的那句：“你什么时间方便，就安排在什么时间，在你能用RSS Reader的地方”，很有点那个立志用曲别针换房子的青年的意味，所以最后还是去了。

起先以为他们是要给新的阅读器产品做用户分析，后来才发现他们关注的其实并不局限于RSS阅读器、浏览器之类的用户行为，他们更在乎通过各功能的操作分析用户的思想，了解用户的阅读行为模式，最后总结出通用的用户行为，可能应用在商业软件的设计中。

参加这个访谈还有一个好处，就是发现自己很多选择上的动机，也使我开始去思考自己的行为模式，这在访谈之前是没有想象的。

最后，我也来帮忙推广一下，如果你想了解一下自己的阅读行为，想为RSS阅读研究出力，而且不太讨厌微软，可以参加他们的RSS阅读习惯调查，中文点这里，英文点那里。

回：建一个动漫网站，能不能挣到钱？

原文回复在http://free.donews.com/5g，回复一个老大的帖子，发到这里纯粹是用做记录。

-——————-

动漫产业的基础应该是动画片的播出、漫画书的出版，这样才能促进核心的产生：形象和版权，有了形象和版权才可以进行衍生产品的开发，才能获得广告以外的其他收入。

现在作国产动漫是趋势，印象中广电为了国产动漫已经快把国外动漫给卡死了，很多名动画片不能播放，能播的也会有各种限制要求。

其实动漫产业应该从代理开始，多吸收其他国家动漫产业的经验，等产业链、资金链建好后再开始自主研发，但是国内公司往往不会爬就先想跑，一下都去做原创，不知道在没有什么经验的情况下能做出什么好作品来。

动漫网站都已经是比较靠后的产物了，如果想做得长久就要深处一个动漫产业链里面，现在的很多动漫网站是经不起版权推敲的。日本的动漫产业很发达，而且日本的版权保护做得很到位，致使日本没有什么动漫网站门户、资讯网站。

又又被点名了，这次是读书

请原谅我标题里面结巴了一下，这次是真的很郁闷，为什么每次都点我？要我一个不大爱读正经书的人怎么写这个题目呀，零零发？还好我一直用豆瓣，能对付一些~

**1.一本你不只读了一次的图书
**《大唐双龙传》，经过了金庸、古龙、梁羽生、倪匡的小说特点，最后发现还是写作时间最近的这部小说最对我的胃口，黄易的大多数小说都有这种现代感，所以像《大唐双龙传》我就读过好多遍。

2.一本你如果身在沙漠时想读的书
这个时候估计得看励志类的书了，我喜欢《基督山伯爵》，不过为了活下去，我更喜欢带着工具书，也许是一本《生存手册》。

3.一本让你发笑的书
《悟空传》，真是一本随便翻到一页看，就想笑的书。

4.一本让你哭的书
《第一次亲密接触》，第一次让我了解到网络作者的书，看到后面真的有点想哭，这是让我最接近哭的书了。

5.一本你希望是自己写的书
无

6.一本你希望从未写就的书
《哈利·波特》，这个童话，世界观我觉得不错，很现代，又很玄幻，完全可以继续写，最后还可以改编成肥皂剧。

7.一本正在读的书
《魔鬼经济学》，问你个问题，给你一个备选答案列表，逐条说明然后否掉他们，接着告诉你正确答案是一个好像没关系的东西。

8.一本读来有意味的书
《引爆流行》，是我读来最有意味的书，感觉就像是有些事情见过，隐隐约约好像知道，但是真让说出来又不知道怎么开口，看《引爆流行》就有这种感觉，而且读完以后能知道是怎么回事。

9.一本改变你一生的书
《BASIC语言程序设计》（具体名字记不清楚了），这是我看的第一本讲编程的书，让我一下进入了这个领域，直接影响我高中的分班，大学的志愿，直到工作还能用到或者说借鉴到编程语言的思想。

10.点名
还有谁？（这句话应该用冯小刚在《功夫》里面的音调来读）这是最头疼的事情，现在已经不知道，跟我有点联系的Blogger还有谁没被点名。

糟糕的用户体验一例

这个糟糕的用户体验来自一个洗手间，它位于世纪金源的上岛咖啡。最近有时会去那家上岛咖啡，因为离洗手间比较近的缘故，观察到大多数人在洗手间外都会产生困惑。 场景1：有人到了洗手间外面，看到关着门，就使劲拧门把手，发现怎么都打不开。有的人就问服务员：“服务员，这个门怎么打不开？“回答：“里面可能有人”；有的人就选择去外面找，心想：“这地方怎么搞得？连个厕所都不能用”。 场景2：洗手间没人，有人到了洗手间外面，看到关着门，宁可等在门外也不愿意去拉，心想：“里面的怎么还不快点？” 场景3：两个男的同时走到洗手间外，一个男的说：“反正里面有两个位置，要不我们一起进去吧”，另一个男的说：“算了，还是你先进去吧，我怕别人说我断臂”，前面的人”……” 一个人不能代表什么，大多数人都这样就有问题了，肯定什么地方不对劲。 归纳一下，有两个问题： 一、洗手间只有一间，为男女共用，每次都要锁门，很不方便。 二、洗手间缺乏足够标识，来表明当前的状态，是有人还是没人？ 咖啡馆之类的地方，通常是会喝茶、咖啡、果汁等饮料，想想看洗手间有多重要？正所谓人有三急，如果在你急的时候不能解决，你会怎么样？ 提醒一下开茶馆、咖啡馆的朋友，别在洗手间这种地方省，会连用户的好感都省了。

开始试用Firefox 2.0

近两天比较郁闷，因为我的Firefox 1.5不能正常访问Gmail了，而且没听说别人出这个问题。我可是一直用Firefox访问Gmail的，这件事一定要搞定~

今天Gtalk上问Keso大哥能不能正常访问Gmail，他说能，忙问版本，答曰：2.0，恍然大悟，马上去下了一个Firefox 2.0b1的中文版，装上后马上连Gmail，结果正常。

之后下了Firefox Portable 2.0 Beta 1，是个英文预览版，凑合先用了，因为没在Mozilla找到中文版的zip包，就用之前安装 的2.0b1文件替换了Portable自带的英文版，好了，中文版的Firefox 2.0 Portable搞定了。

2.0b1的支持的扩展还少得可怜，强行装了几个低版本的扩展，都以失败告终，正常启用的扩展（以后随时增加）：

  • 365key 天天网摘右键收藏 0.2.1 这可是2.0专用的
  • Adblock Plus 0.7.1.1
  • CustomizeGoogle 0.49
  • DOM Inspector 1.8.1b1
  • FlashGot 0.5.9.995
  • Gmail Notifier 0.5.6.3 有了它我就安心多了
  • MR Tech Local Install 5.2 这个还是一如以往的好用
  • NoScript 1.1.4.1
  • Organize Status Bar 0.3
  • Pearl Crescent Page Saver Basic 1.2
  • Searchbar Autosizer 1.1 兼容，但是没见起作用
  • SearchStatus 1.16
  • Tab Mix Plus 0.3.0.60805 是测试版，支持2.0，来自Tab Mix的论坛

毕竟这个版的Firefox还是测试性质的，有很多地方不太令人满意：

  1. 兼容扩展太少，如上，不说了
  2. 发现Firefox 2.0b1这个版对Ajax的支持还不好，像我的Wordpress后台特效全没了，隐藏的菜单都显示出来了，看来除了之前出来的IE 7.0要适配，连Firefox 2.0也要配一下了~
  3. Feed全被Firefox接管了，只要是Feed，一律都由Firefox的内置方案渲染，根本不管是不是经过Feedburner、Feedsky加工过的
  4. 搜索栏感觉设计得不好，不如1.5的人性化

Update：多亏Keso大哥提醒，借助MR Tech Local Install 5.2，很多扩展都装上了，乌拉~
