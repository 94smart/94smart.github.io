胡扯一下论坛与社区

BBSBBS社区社区网站随笔随笔

最近看麦田又开始说社区了，这次说的是贴吧（1，2），从使用流程、管理权限、用户需求、推广等方面入手说明用户为什么用贴吧，看完很有感觉，也来小小扯一下论坛有关的。

论坛属于最基本的社区功能，但并不是孤立的，在社区里论坛相当于骨架，可以很容易结合个人空间、交友等其他功能。

其实论坛有多种形态，包括常规论坛、讨论区、贴吧等。常规论坛是最基本的，板块内容、结构是由站方管理员控制的；讨论区一般是由所附的产品、兴趣、问题等发起的特殊论坛，管理权限介于站方管理员与高级用户之间；贴吧相对自由，可以由用户自行创建、管理，所以从这点讲，贴吧最Web2.0了，看上去也最UGC了。

这些形态都是论坛发展的阶段产物，同时也是论坛逐渐开放的结果，对于用户来说就是管理权限从网站经营者向用户逐渐移交，但这些形态又不互相排斥，并不是一个存在另外两个就消失，而是可以同时存在的，为不同的需求去服务。

而且如果用流行Web2.0理论来说，论坛从头到尾，且不管现在有多少形态，它一直是Web2.0、是UGC。

《长尾理论》读书笔记（一）

互联网互联网电子商务电子商务随笔随笔

最近正在看《长尾理论》，虽然试读本很烂，我还是第一时间买了，读起来还不错，是关于互联网的经济理论及实践。

读完前五章，按照自己的理解汇总一下：

  1. 互联网的长尾市场
    网上商城的"货架"可以数百倍、数千倍的多于实体店铺，使无数个线下放弃销售的低销量产品，通过互联网增加了销量，这个市场仅在互联网上才有，也是长尾市场。
  2. 大热门将被无数小热门取代
    小热门也是长尾，随着互联网的发展，越来越多的小热门出现，导致依赖传统模式的大热门衰落。书里并没有说要被取代，只是我这么觉得。
  3. 互联网使产品制造、传播、搜索成本降低
    互联网和PC等工具的普及，使产品更容易被制造出来，也更容易传播，同时使用户的搜索成本进一步降低。
  4. 业余和专业共存及用户创造内容
    举了维基百科的例子，爱好使无数业余人士乐意去创造内容，并不比专家产生的差，且绝对比专家的全面，这里我觉得用草根和精英更合适。
  5. 电子商务的数字化市场及库存、物流成本的降低
    用亚马逊的模式可以说明，有形产品方面，与大供货方的合作可以大幅降低库存和物流成本；数字产品方面，库存和物流成本更低，甚至可以无限趋近于零。当这些成本如此之低的时候，这个市场将随产品的丰富而不断扩大。

注：现在只用上下班在地铁上看，看得比较慢，所以后边的也会写得比较慢。

正在验证你对"94Smart’s Blog"的拥有声明。一般这个过程需要几分钟，请耐心等候。验证成功之前请不要删除包含豆瓣验证码的那篇文章。

话说回来，同样是归属验证，我觉得Google Sitemaps处理得就很好，用网站根目录的文件解决的，这既解决了验证问题又不影响网站的用户访问。

豆瓣这次的验证比较奇怪，生成的验证号一定得发到Blog上，而且需要确定RSS已经显示，才可以继续执行，看得出来使用的是RSS内容分析方法。虽然提示验证后可以删除，但是终归会让访问者看到令人困惑的内容。

当然豆瓣可能考虑到有些用户使用的是BSP的空间，自己不能上传文件或修改页面，只好使用RSS验证，但豆瓣有没有考虑过某些Blog的RSS功能关闭或者损坏的情况呢？数量太少不用考虑了？

还有，我挺奇怪的，自己的Blog地址明明在豆瓣上的个人信息填过，为什么还要在自己的Blog上声明？与其这么费劲的做一个不太完善的验证功能，还不如简单点，相信用户提交的信息算了。

ReviewMe.com的付费评论

BlogBlogIM网站评论评论随笔随笔

The following is a paid review. 这是一篇枪稿，如果您厌恶这种内容请给我留言，也可使用Gtalk联系我。

ReviewMe这个付费Blog评论平台没出来多久，就被无数国内的Blogger炒起来了，alexa排名在几天内上升了1,265,026位，写这篇的时候已经到700了。

我之所以写这篇，主要是因为看了herock、zola、shunz、doubleleaf几个人写的，觉得可以试试，但并不指望它能给我带来什么收益。

这个网站很新，但是模式其实已经不新鲜了，之前也看到有做的，简单来说就是你给钱我写Blog枪稿，我所关注的是广告主怎么去评估、考核这些付费内容，以及平台的组织方法。

口碑传播现在已经越来越受重视了，Blog正好是一个不错的工具，想象一下这样的情况：一个产品即将或者刚刚面世，广告联盟的Blog充斥了这个产品的大量评论，对产品的推广是不是有积极的效果？我觉得多少有点，但是这还得要求其联盟的足够强势。

ReviewMe深知这个道理，所以刚上线第一个推广的产品就是自己的网站，动用了2.5W美金，应该能达成他们的初步推广目的，我觉得对这样一个平台推广费用已经不少了，Blogger参加的要求很简单：

  1. 文章开头说明该文章为付费推广
  2. 字数不少于200个英文单词

推广评估这方面，我觉得ReviewMe并没有作出太好的模式，应该只是综合网站排名、反链等对Blog进行估值，有点一刀切的感觉，不知道会不会影响Blogger参与的热情。

不知道这种模式在国内是不是也可行，但是不管机会是多么的小，国内都会马上有一批复制其模式的平台、联盟，看着吧。

p.s.我的Blog还值60$呢，一篇文章也有30$，不知道能不能拿到手。

BS免费赠阅的《长尾理论》试读本

IM评论评论读书读书闲言闲言随笔随笔

《长尾理论》这本书是早就想看的了，偶然发现dangdang有试读本可以看，也就不管三七二十一，按照条件申请了，中间也不惜跟他们客服费劲了两次，可是得到的是一本残缺不全的小册子，太令人失望了。

首先，必须说这个试读本绝对不是免费赠阅：要想索取这个试读本，首先要去dangdang选购一本中信出版社的书（最低也得差不多20），才能在购物车页面的左边"特惠区"点到小得不起眼的链接"超值赠送区"，在打开的页面最下面才能看到将试读本加入购物车的选项，最后还得付运费5块，所以这绝不是"免费赠阅"。

其次，试读被出版社理解为，内容不全的选择性阅读，薄薄的四十几页纸，一共才提供前五章的部分内容，就这部分内容还打"马赛克"：涉及该书的中心思想一概给你两句话，大意是：“XXX此处欠奉，欲知详情请阅读XX出版社即将出版的《长尾理论》"，靠~

基于以上两点，我只能BS这个活动的策划了，整个一个趁火打劫，别人不敢说，我以后是不会再上当了。

我觉得吧，这试读本还不如不提供印刷版呢，放在网上发布多好，给你看点节选，不想给你看得内容就不发，多节省成本呀，我说的是出版社和书友的成本。
