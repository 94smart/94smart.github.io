黑莓OS6刷机最简方法

最近想更新自己的BlackBerry Bold 9700，看了好几个教学帖子，要选择ROM、WIPE、改什么文件名、输入PIN等等，感觉比iPhone越狱还麻烦，不过昨天偶然发现了最简的方法：

  1. 更新BlackBerry Desktop Software到最新版本，我的是4月25号发布的6.0.2.44，100M出头
  2. 更新后连接手机，进过若干同步过程，主界面会显示软件版本5.0 Bundle xxxx，下面会出现有新的更新软件版本6.0 Bundle 1879之类的，选择更新
  3. 接着自动备份、下载新的ROM，这个时间比较长，可能是直接从外网下，我等了5个小时
  4. 下载完，后边都是自动更新，over

以上方法仅限9700，我的T-Mobile版是刷成功了，不知道其他运营商和混刷的能不能用。 更新完成后，需要自己设置、主题、处理低版本的应用等等，我发现很多OS6不如5的地方，比如输入法切换等。

未来会不会有互联网版数码相机？

先来几个问题：

  1. Instagram类的移动产品太多了，是否说明有足够强的用户需求（比如特效和分享）？
  2. 智能手机在取代卡片型数码相机，数码相机、单反可否向智能化、网络化发展？
  3. Android等OS对移动设备的定制能力很强，以后会出现用它定制的数码相机吗？
  4. 5年后WIFI、3G、LTE啥的会大行其道吗？

如果上述都是肯定的，未来会不会有互联网版的数码相机呢？ 试想下，用Android定制联网版数码相机或3G版单反，拍摄质量远超手机、效果堪比单反，拍完可加特效，然后直接上传网上相册或发到微博，这得圆了多少单反控的互联网梦呀～ p.s.也许我孤陋寡闻，不知道这玩艺已经有人做出来了。 update1：eye-fi 这种特殊的SD卡可以把数码相机拍下来的照片通过wifi传送给iPhone、iPad，最起码解决了无线传输照片的需求，直接分享还不行。
