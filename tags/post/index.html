2012-03-20 广告让手机更费电？

android post 广告

先举个例子，电视会耗电吧，那电视广告耗费了不必要的电能吗？ 不过，有人会认为手机广告会消耗额外的电量，事情是这样的：

由普渡大学和微软组成的研究小组发现，Android app相关的电池消耗75%是由发布广告导致。 受测的流行应用包括愤怒的小鸟、FreeChess以及纽约时报app，受测机器是HTC Passion （Nexus One），运行的是Android 2.3。测试愤怒的小鸟和FreeChess的电池使用情况，结果发现游戏本身仅占app耗电的30%不到，其余70%均为上传用户信息指标、位置以及下载和显示广告所消耗。

2012-03-01 两份广告市场的数据

post 广告

最近看到两份关于广告的数据：

  • 2月29日消息，据国外媒体报道，据市场研究公司StrategyAnalytics最新发表的《全球广告预测》报告称，2012年全球广告开支预计将增长4.9%，达到4655亿美元。2012年全球在线广告开支将增长12.8%，达到832亿美元。以奥运会为首的对全球产生重大影响的事件、美国总统大选、欧洲足球冠军杯赛以及日本从地震中紧急复苏等事件将为2012年全球广告开支描绘一个更明亮的画面。我们预计2014年全球广告开支将达到5000亿美元。
  • 2012年2月27日昌荣传播对外发布了《2011中国广告市场与媒体研究报告》。报告指出，2011年中国广告市场继续保持平稳增长，以刊例价计算，广告投放总额为6693亿元，比2010年增长14.5%，高于GDP的9.2%增长率，低于昌荣传播在2011年初预测的15%的增幅。2012年中国经济增幅继续趋缓，房地产、交通、家电、食品等行业的广告投放更会趋于保守，而金融、饮料也会趋于平稳，由此，预计2012年中国广告市场会低于2011年的广告增长，维持在13%的增长率。

看预测数值，差不多2012年全年的广告开支可以用来买下苹果了。

2012-02-27 移动广告正在发展进行时

post twitter 广告 微博 移动广告

前几天新浪微博banner广告的报价，真是一石激起千层浪，有骂有捧，真够热闹的，不多说了，查看那条微博可以看的更仔细。 商业化的路上总有那么点难过的，微博也不例外，坐拥号称3亿用户的核心产品，变现一定是个长久的话题，广告应该只是一个开始吧，其他模式肯定也在搞，手机微博广告的开卖也挺正常的：第一，新浪历来是以品牌广告为主要收入的公司；第二，手机版微博顶上早就有Banner了，开出来的位置是一定要卖的（好歹也号称40%以上是移动版用户呢）。 基于社区、沟通的营销、广告形式挺多，twitter这个微博的先驱也正在研究各种形式，到现在也是有输有赢，Banner这种硬广确实和微博的内容呈现形式有一定不和谐，但没有广告主尝试、没有品牌成功案例，后边什么营销都是扯淡。 不过话说回来，移动广告还是大有可为，除了继承互联网广告模式外，还有移动端天然的优势，基于移动设备的GPS、光感、摄像头等交互，另外就是真的关心用户的使用体验，banner不见得是最好的广告形式（虽然iAd只有banner），怎么让用户得到实惠的同时传播品牌形象，这是另一个要继续研究的课题。 移动广告还在发展过程中，好的开始可能就是成功的一半，现在国内主要的移动门户新浪、百度、3G门户、凤凰等都在发展自己的广告模式，应用内置广告的平台现在也有个几十家，愿意内嵌广告的应用有上万了吧，活跃用户怎么也得过几千万，其实国内现在最缺的还是成功案例，和品牌的深入合作加上适当的交互也许可以迎来美好的未来。 p.s.本人现投身移动广告行业，公司需要招聘若干视觉设计、文案策划、前端互动设计，工作地点北京东四环住邦2000，有意的可以Gmail联系我 @94smart

2012-02-16 页面制作，不，是前端

CSS HTML5 Javascript Job post 招聘

  • 多少年过去了，做网站还是得靠页面制作，不管是CSS还是JS，现在还得搞HTML5，只是职位改叫前端了；
  • 用CSS+JS代码完成针对屏幕尺寸的页面适配叫响应式设计了，前端更成了很多产品的命根子；
  • 原来以为WAP网页WML、2.0暂时用不着JS了，结果Jquery Mobile之类的来了，说：移动互联网我们也能动起来；
  • 现在有了PhoneGap和钛，用HTML5+JS就能搞手机应用了；
  • 现在有Node.js之类的，可以用服务器端运行JS搞些东西了；
  • 很多时候设计师还是只能望代码兴叹，最后也得逼自己学这学那的；
  • Javascript语法成了更有发展的玩意，这不仅讽刺而且夸张，但如果你是技术或设计，不懂一点JS，还真不好意思说自己是做网站的。
