最近用的小玩意们

Google桌面软件桌面软件网站随笔随笔

最近用了不少轻量级软件、网站，挑几个觉得还可以的推荐一下：

GTD相关的：

  • Remember The Milk：“记住牛奶”，我觉得名字挺好玩的，清爽设计配合AJAX操作，用着很舒服。虽然早就注册了，不过今天登录的时候才知道它已经有了中文版，这下用得更爽了。
  • Google NoteBook：最近刚刚支持开放笔记，而且Firefox的扩展好像优化了一下，比以前占用的内存少了点。
  • EverNote：一个功能超强的Web/文档摘录、剪贴、记事簿，可以根据时间、类别进行区分，支持多种模板，可以用来做GTD的Todo等操作，支持IE和Firefox扩展，最可惜的是不支持中文。
  • Task Coach：一个小巧的开源日程管理工具，功能已经够用了，不需要去用大块头Outlook了。
  • Folder Marker：给文件夹换图标、颜色的工具，可以标示各工作文件夹的优先级等。

Firefox扩展：

  • del.icio.us Complete：del.icio.us的管理器，可以将del.icio.us放在Firefox的侧栏上，支持标签列表、标签云、书签列表等，摘录的界面比del.icio.us官方提供的要舒服点，用着这么舒服以至于我把官方扩展都卸载了。

其他软件：

  • Website-Watcher：网站更新监控软件，可以根据关键词等条件监控网站是否更新，同时支持不怎么样的RSS阅读，我用它来监控Zotero（又一个Firefox的摘录类扩展）是不是开始提供测试。

当然还有很多，不过有些用了两下就没兴趣了，有机会下次再说。

顺便祝长期关注我Blog的朋友国庆节快乐、中秋快乐！！

Opera 9，可以用用

IM互联网互联网桌面软件桌面软件浏览器浏览器用户体验随笔随笔

Opera 9发布几天了，下了一个试了试，还不错，感觉Opera在用户体验方面做得越来越好。

Opera 9的一大亮点是支持组件（Widgets），初看上去是学FireFox的扩展，其实不只这样，Opera还参考了Yahoo和微软的桌面组件，将很多娱乐功能搬到了浏览器里。

与FireFox的扩展相比，Opera的组件并不集成在浏览器上，而是单独开一个小窗口，可以与浏览器分开显示。昨天发现一个不错的组件：Pandora Radio Widget，有了它，就可以将Pandora的大窗口缩成一个小条的播放器面板，爽~

我还关注的一个功能是BT集成，宣传文档里面说从此不再需要其他的BT客户端软件了，用Opera就可以搞定，我觉得Opera是浏览器产品里比较有前瞻性的，现在BitTorrent都与正版发行商合作了，以后的发展应该更好，这样做不管从哪个角度说都是正确的，不过让我想起了微软和IE的做法……

Opera 9还有很多新增功能，在此就不赘述了，不管喜欢IE还是喜欢FireFox的都可以去试试。

最后扯两句，Opera自从宣布免费以后，在社区应用方面的发展很迅速，走了一条与IE、Firefox都不大相同的浏览器之路，即软件、服务、社区的集成，也许这种模式只有IE的后续版本依靠live.com可以去打败。扯完~

貌似Sogou输入法还不错，但我还是用不惯

IM互联网互联网桌面软件桌面软件输入法随笔随笔

上周在大批Blogger推荐的情况下，我终于使用了Sogou拼音输入法。 我感觉优点在于新鲜的词、热词一打就出，速度还特别快，而且支持多种键位用法，看来他们是下定决心要挖其他输入法的墙角了。 因为我用的输入法比较单一，只用微软拼音2003，不知道它和其他哪个输入法比较像，只知道它和微软拼音操作不同。 所以它还是不适合我，原因如下：

  1. 输入拼音后按空格或数字选词后就直接打出来了，不能像微软拼音一样需要用回车确认。这一点可能别人认为是优点，而我认为是缺点。
  2. 微软拼音支持拼音的自动拆分，Sogou不支持。比如打"xian"，Sogou候选词里面只有字，而没有词，事实上我想打的是"西安"，要想达到这种目的必须输入"xi’an"。
  3. 输入法的显示条怎么隐藏？我到现在也没用出来。

还有几个小地方，就不一一列举了，我觉得，像我一样习惯微软拼音的人可能很难接受Sogou。 但是我还是推荐使用紫光拼音和拼音加加的朋友试试Sogou拼音，没准你会喜欢~ 附我的想法变化： 第一印象：Sogou也可以做拼音输入法？ 第二印象：Sogou的拼音输入法还不错 现在：Sogou的拼音输入法不适合我 p.s.经KESO一提，想起来以前还在学校里做过智能狂拼II的测试工作，起先感觉还不错，后来发现其与Windows平台上的程序兼容性极差，才知道这种与系统相关性极高的东西只能用微软的。。。。。 Update：下午刚写完这篇，晚上就看到Sogou输入法将继承Sogou Toolbar的流氓软件身份，在未来的日子里安插在各位的PC上。对于这篇blog，我不知如何是好，就留着吧，懒得删了。 Update 2：今天（6月18日）才注意到新浪的首页有Sogou拼音的广告，其中有个链接还连到我这里 Technorati : IME, Sogou

Zoundry新版进化

apiBlogBlogIM桌面软件桌面软件随笔随笔

几天没在，Zoundry就出了个新版，这版跨过了1.0，直接出了个1.0.23，修正了一批Bug，还推出了一些新功能，看来Zoundry还想将Beta继续下去，我原以为过了1.0会把Beta撤下来~

看看都有什么改进：

  1. 拼写检查（词典要自己下载）
  2. XHTML源码编辑及自动完成
  3. XHTML代码规范验证工具
  4. XHTML整理操作
  5. 支持语言本地化
  6. 查找/替换
  7. 提供MovableType API和WordPress文本实体扩展（例如摘要操作）
  8. 增加对Blog实体的排序方法（如按时间、按字母顺序等）
  9. 建立Blog帐号的时候增加支持RSD来识别API设置
  10. 其它修改，包括图片快速对齐菜单，用"Link to Posts"增加旧帖子的链接等

原文：http://www.zoundry.com/releasenotes.html

这个新版改动还是挺大的，主要是增加了源码编辑操作，又向Dreamweaver迈进了一步~界面改动不太大，用起来还是挺习惯的，希望下次改进不要再改动编辑区了，越改越复杂。

最令我感觉人性化的操作是崩溃恢复了，刚才写这篇的时候使用了XHTML代码验证，还使用的是严格式（Strict）的，一点就报错还自己关闭了，然后我就傻眼了，写的内容都没保存，结果再次打开Zoundry，发现内容还好好的在编辑区里，真是高兴死了。

还有就是本地化方面，Zoundry并没有采取其它软件的本地化策略，没有提供语言包，而是提供了开放型的解决方法，让用户去自己解决，想改什么就改什么。不过我觉得这种方法有点取巧，虽然很快就会有用户自己定制的语言包出来，但是却让大部分用户决得不够方便。

总的来讲，Zoundry在进化，但是方向并不是一直朝Blog编辑器去的，而是向专业网页编辑器（Dreamweaver），或者更夸张点说是向专业文本编辑器（Word？）去的，越做越高端，不清楚是不是要走高端路线了。

不过话说回来，这个编辑器还是值得继续推荐的，快下手~

2005年最系列 - 最常用

FeedGooglesnsWeb2.0Web2.0桌面软件桌面软件网站评论评论随笔随笔

最近由Tangos带起了一连串的2005之最，看得挺起劲，不过别人的感觉毕竟和自己不同，所以来写个2005年最系列，要说明的是数量随机，且不区分网站和软件。 今天先写最常用的：（按使用频率排序）

  1. Maxthon 每天上网必不可少的，也是最主要的浏览工具，自从去年发现后一直用到今天，易用性一流，不过IE的糟糕内核决定终有一天会被我换掉，候选极有可能是FireFox。
  2. Gmail 现在是我的主要交流用信箱、文件服务器、消息提醒等等，好用的不得了，真很难再找出一个和他一样的了。
  3. FeedDemon 我的主要信息来源阅读器，虽然我试着将Feed都传到Bloglines上，但是还是不习惯Web的阅读方式，更喜欢客户端的阅读方式。
  4. 365key 我每天主要的阅读记录全在这里， 且为我的Blog增加了昨日收集这样的每日固定内容。
  5. Drupal 我Blog的建站程序，功能强大，但是有时觉得操作太复杂了，明年的备选将是Wordpress。
  6. Zoundry 虽然这个软件是最近几个月才开始用的，但是用过后就再也没有放下，这个软件的几次关键更新更坚定让我接着用下去，这里特别感谢Herock，没有他的介绍我很难知道还有这么好的Blog软件。
  7. FireFox 最近使用频率越来越高，丰富的插件以及DIY浏览器的感觉都在吸引我，用他可以将我的大多数应用整合在一起，如果能在人性化方面参考一下Maxthon就更好了。
  8. Flickr 今年发现的UI最棒的Web2.0站点，用的太舒服了，不过可惜我不是摄影爱好者，没有好看的照片放上去。
  9. FeedBurner和FeedSky 这两个站上都有我托管的Feed，我也要经常上去看看统计，前者经常推出好用的小功能，后者的统计很详细，两者不分大小，不过在国内我更看好Feedsky，因为FeedBurner宣布和Bokee合作，坏了名声。
  10. Del.icio.us 主要的英文文章阅读记录，替代了年初的Furl，英文不大好所以这个用的也不多，明年会努力多使用。
  11. Orkut 为什么会有他？因为自从在我Blog上公开发放Orkut邀请之后，每天都要上去发邀请，同时也可以长时间观察这个Google的SNS，最近又有些小改动，有兴趣的可以去看看。

一共11个，为什么会出来这么诡异的数字？ 占少数的软件和占多数的网站组成了我的网络应用，将他们分开会看不到真正的趋势，且有很多是组合起来使用的，所以不能分。 p.s.定期回馈是件有益的事情，但是一年好像太长了~
