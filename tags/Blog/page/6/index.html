2007-09-11 杂谈：对Blog的想法

Blog Blog Blog AD Blog Network Feed Wordpress Wordpress 随笔

那天在WordCamp Beijing 2007，曾跟Herock聊了些关于Blog的话题，回来想了想觉得挺值得记下来的：（有所扩展）

  1. Blog或者说博客的概念现在应该已经不重要，只要你愿意，愿意把Blog做成什么都行，也可以把任何Log形式变成Blog。
  2. 想写就写，不想写就不写。
  3. 想写什么内容就写什么内容，多写对自己有用的，最好同时还对别人有用的。p.s.看MR.6写的后，我越来越想写有点意思的东西了。
  4. 需要对写出来的内容负责。
  5. 广告愿意放就放，愿意放哪儿、愿意放谁的都随自己。p.s.个人感觉广告只要不多到像某BN那样，估计看的人应该都能接受。
  6. Blog Network这种形式的Blog群，很有价值，而且专业和松散类型的会越来越多。
  7. Feed需要彻底平民化，用普通网民可以接受的概念去包装，需要Feed托管商、内容提供商（或BSP）和阅读器厂商共同努力去推广。
  8. Wordpress页面使用动态显示，虽然可以通过URL重写成类静态地址，但是访问还是不如MT直接生成静态页面的速度快。
  9. Wordpress还是不够傻瓜，每次升级都得折腾一阵，如果不用直接在服务器上SVN也可以自动升级那该多好~

2007-05-16 升级Wordpress 2.2，很顺利

Blog Blog PHP Wordpress Wordpress 随笔

Wordpress 2.2版今天早上发布了，我也赶个新鲜，第一时间升级了我的Blog程序。 过程很顺利：

  1. 备份了一下数据库
  2. 备份几个文件，.htaccess、robots.txt和wp-config.php，以及我的主题、扩展
  3. 把其他文件全删除
  4. 下载Wordpress 2.2版程序包，解压缩，上传到相应位置
  5. 修改wp-config.php，设定define(‘DB_CHARSET’, ‘utf8’);
  6. 去浏览器执行http://youhost/wp-admin/upgrade.php

然后OK了，这是我升级Wordpress这么多次中最轻松的一次~以前因为编码、预览问题还要改程序代码，今天发现这些都不用了，因为Wordpress把这些都改好了。 用了用，感觉跟之前的版本没有太多的差别，倒是觉得反应速度稍微快了一点。 我一直很纳闷，为什么Wordpress不出一个自动升级程序，每次都让用户去替换、修改如此多的文件实在是太不方便了。

写在前面：统一概念BSP=Blog服务提供商，BCP=Blog内容提供商。 以前有人说过，互联网的发展每几年就要轮回一次，中国互联网发展到今天已经足够一个轮回了。 在轮回里面充满着新奇的点子、大把的票子、勇敢的疯子和跟风的傻子，不管过程是多么曲折离奇、柳暗花明，结果就是疯子的点子烧掉了傻子的票子，也烧出了门户网站、电子邮箱、即时通讯、搜索引擎、个人主页等应用，最终确立了ICP的内容为王理论。不能说ISP死了，只能说ISP已经成为ICP的附属了。 废话说得差不多了，该说正题了。 近一年到两年的时间，Blog得到了广大网民的认可，开始爆炸式增长，用户人数据说已经突破200万了，这也造就了国内BSP的发展空间。三大BSP（BC、CB、BB）现在都已经“人满为患”了，如果不是分别找到了VC，有充足的资金，恐怕光数据存储和带宽这块就已经够关门的了。 这也会让人联想到上个世纪末、本世纪初的网络泡沫，电子邮箱、主页空间、即时通讯等免费服务的提供，因为用户增长过快，不得不投入大量人力物力去维护不挣钱的服务，很多公司的资金也是这样被蒸发的，最后许多免费服务都消失了，取而代之的是收费服务。 BSP也许也会走这样的路，当BSP无法忍受不能盈利的负面影响的时候，现在的免费服务，也是用户最需要的服务就会消失了，毕竟存储用户的资料、网志也是有成本的。但是BSP这种盈利模式不清晰的行业如何真的摆脱这种困境呢？我认为BCP是可以弥补这方面缺点的。 首先，BCP可以独立存在，硬件和软件投入都比BSP少很多； 其次，BCP的目标应该是明确的，他是提供内容的，以内容来盈利（比如广告）； 再次，BCP可以促进BSP的发展，那种生长在BSP下的BCP可以为BSP提供源源不断的注册用户。 Weblogs.inc是最成功的BCP模式之一，他的运作模式是签约职业Blogger，然后用内容去换取广告费，其旗下现在已经有超过70个专业的Blog，真正称得上是个“Blog矩阵”，同时每月的Google Adsense广告费已经超过1万美元。 国内如果有这样的BCP，只要能聚集一些keso、topku、chedong、王翌、wangjianshuo这样的Blogger，盈利应该不成问题。 还有一类是Boing Boing这样的Blog在线杂志，由一个或几个人维护，在上面总能看到有意思的内容，类似模式的还有lifehacker、Tech Digest、The Bag Lady、Slashdot等等。 除了以上模式还有博客中国的门户模式，虽然路子有点窄，但是不可否认确实是一个当前不错的模式，可以把BSP的成果都转变为BCP的资源。 说了这么多，不是想说BSP不被看好，而是单纯的BSP没有什么前途，只有综合了BCP的优势才能有长足的发展，希望现在的BSP能看到自身的缺点，赶快进行调整。 我更看好纯BCP的发展方向，因为内容为王的路线在短期内还是不会被替换的，当然融合了BCP的BSP也会有很好的发展。 暂时想到这么多，以后有什么再说吧。

2005-01-27 Blog的另一种形式-ChatLog

Blog Blog

经过一天的工作，我想大家都累了，特写一篇消遣性文章让各位YY。 看了Topku的有了瞎话你就喊，突然产生了一种奇怪的念头，想到了Blog的另一种形式——聊天记录型Blog。 Blog本身就是一种记录，Msn、QQ等即时通讯软件的聊天记录也是一种记录，这本来就是共通的。试想，一件事情讨论/达成共识的过程，对参加讨论的人是多么有意义，这比整理后的结论性文字更能启发大家的思考。 这种形式的Blog其实早就有了，记得以前就在某论坛看到过有人把争吵的聊天记录贴出来，看的人虽然RP接受极大的考验，但是获得的信息是连贯的，这也符合Blog以话题为导向的特点。 形式挺好玩的，具体操作的时候如果有必要可以把人名/网名都改掉，替换成张三、李四什么的，也不会让记录里面的人太不好意思。 这种模式可以很好的商业化，实现起来我想也不会太难。举个例子，哪天腾讯给QQ用户一人一个Blog空间，专门公开他们的聊天记录。这个可不是随便说的，现在QQ什么的聊天软件都有上传聊天记录的功能，指不定哪天我们的QQ记录就被腾讯发表在什么地方了。:-) 今天的胡说八道到此结束，各位看客我们下次再说~
