唠叨：淘宝退款操作的疑似BUG

之所以说是疑似BUG，是因为没在淘宝帮助里看过在Mac上能做退款，但也没像数字证书一样说明只在Windows的IE有效。 事出有因，最近公司给配的DELL笔记本坏了要维修，于是就拿出雪藏半年多的Macbook办公，又因为放在散热支架上键盘按着不舒服，于是跑到淘宝上拍了一个G5键盘（使用家里PC操作），寄过来发现用处最多的空格键没法用，跟卖家协商后决定退货、退款。 退款申请得卖家同意，才能叫快递退货，所以现在的情况就是我只能晚上在家申请退款，在公司叫快递发货，结果卖家有修改意见，想马上在Safari改吧还不行，修改退款协议页面下方一片空白，没法进行下去（见下图），只能等到回家操作。 退货这么一件小事，以前接到货就能马上操作，现在变成要牵扯3天以上才能解决，阿里好像也不太带进MacOSX的用户呀（虽然现在已经可以正常登录了）。 牢骚发完，想问一下，阿里的哥（们），啥时候才能在Mac上踏实的使用淘宝和支付宝呀？ 也许阿里的哥会回答：不要迷恋哥，哥只是一个传说！

近期随录及预告

预告先：由于本人近期私人事务太多，下周起每天的网摘会受影响，不能保证天天都做，在这里先通知一下，恢复时间估计得到7月下旬。 近期随录：

  1. Gravatar，上周在我的Blog上加了Gravatar头像插件，只要在Gravatar注册过的朋友在这个Blog留言，同时使用的还是注册Gravatar时的邮箱，就可以在留言里面看到头像了，没有的可以到gravatar.com注册一个试试，记得将头像设为R级以上，我的Blog屏蔽R级以下（X级）的头像显示。（我自己的头像为什么看不到？）
  2. ClickTale，终于收到邀请可以测试了，大概试用一下感觉功能还可以，但没有CrazyEgg的视觉图直观。
  3. Access Flickr!，这是为Firefox无障碍访问Flickr的插件，好用得不得了，虽然也可以使用greasemonkey脚本，但是还是觉得插件更方便，没装的赶紧装吧。（正琢磨买单反呢，结果flickr不能用，那叫一个郁闷）
  4. Safari 3 beta，好像是今天才出的，下了一个用，那叫一个郁闷，默认不支持中文和拼音输入，感觉这个产品不像是给非英语世界使用的。另外，iphone的Web端开发（不只是Ajax）好像也要依靠Safari引擎，现在又出了Windows版，前端工程师有的忙了，也许不远的将来CSS要多写一段适配Safari的。
  5. Axure RP pro 4.6 beta，前两天收到更新简报，说又增加了不少交互操作，例如条件语句、页面切换时的变量附值等等，想了解更多可以看他们官方Blog的介绍1、介绍2、介绍3。
  6. MindManager 7，也出新版了，想要试试还没来得及~
