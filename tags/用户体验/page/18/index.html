我靠，还有这种花店？！

用户体验随笔随笔

听说过这种花店吗？

你进去买花，看了好久，价钱也给你报了，你很满意，然后挑够了数量，写好送花日期、地址、电话、联系人，接着把卡片也好了，并且为自己的措辞感到自豪，末了，店主却告诉你：“不好意思，你挑的日子不对，这些花那天不卖，但是我们在那天准备了特别的花束，还打九折，您要不要直接换了？“看了一下，那天的特别花束还真是贵，就算打九折也令人不能接受，于是乎只好不在这个花店买了。

真是莫名其妙，送上门来的买卖都不作，就算你那天不卖花，你也得提前说呀，非得客人准备好了掏钱，又不卖给人家，还给人推荐别的，这不是等于把客人往外推？有病吧？！

这种花店真的存在，就是著名的网上花店shalala，在这个特别的节日给我留下极差的印象，我决定以后都不在它那里订花。
