2015-03-13 2015再出发

2014 创业 随笔

发现经过多年，现在再写blog都很随意，不在意有什么疏漏，都是随性而来，随笔而成。
2015的3月都过去一半了，我发现（也确实是时候）需要重新整理一下思路，然后重新出发。
过去的一年很丰富、但不精彩。
去年创业开始不顺，我们经历了预备裁员、劝退、挽留、又拿到投资、加薪、展会、押宝、法庭宣判、延期、套牢、减薪、外包、自负盈亏、当顾问……每个月都有不同的事情，每个词都是一段血泪史，确实很丰富。
但真的不够精彩，因为去年很多事都是目标明确但结果极其不理想，干了很多事但是没有什么收获。当时感觉每个项目成我们都能继续下去，但现在来看每个项目都有很明显的问题，资源、合作方能力、项目把控、时间点等等都有各自的不足，但怎么就这么肯定一定能成，试了无数种方式，还是没挣到钱，资金套在了拖期的项目上。
这次参加的创业以管理层停薪留职、员工遣散、公司就剩下个空壳和2个烂尾拖期的项目告一段落。
我很不甘心。
但不管怎样，生活还要继续，还是收拾收拾，然后接着再出发好了，希望下次创业结果好一点吧。

2014-12-31 2014年年底话

2014

这一年过的好快，这一年干了好多事，埋了很多种子，但要很多都不知道什么时候才能发芽、成长再到收获。
今年做了很多以前没做过的事情，工作、外包、顾问……同时跟进不同项目的感觉还行，感觉自己前面几年的工作积累还算有点用，所以以后不能急于否定自己。
个人玩具项目从Python网站变成手机应用，又再变成PHP网站，虽然最后不用了，但在过程中又找回了不少当年的激情。
今年给底自己的奖励是PS4一台，也算是还了个愿。
对明年的祝愿是：再简单一点，再耐心一点，然后想到就去做。
