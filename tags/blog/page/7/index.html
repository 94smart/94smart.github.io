使用Zoundry连接MSN Spaces的方法

本文仿照Dareo的写法，下面是用Zoundry Blog Writer连接MSN Spaces的步骤：

  1. 如果没有MSN Spaces，先去http://spaces.msn.com申请一个。
  2. 登录，修改你的MSN Spaces->设置->电子邮件发布"。
  3. 打开电子邮件发布，见下图（不知道为什么其它选项都是中文的，只有这页是英文的）
  4. 设定一个保密词
  5. 如果没有最新的Zoundry，可以到http://www.zoundry.com下载。
  6. 打开Zoundry，点Tools->Blog Account Settings
  7. 创建一个新的帐号用户名填你的目录名，例如，我使用i94smart，因为我的Spaces是http://spaces.msn.com/members/i94smart，密码是你打开电子邮件发布时设定的保密词。
  8. 类型选择"MSNSpaces"，Zoundry设定很简单不需要设定SSL和端口
  9. 接着点"Next"，直到结束。
  10. OK，你现在已经可以用Zoundry在MSN Spaces上发布你的Blog了。

终于写完了，松一口气~

Zoundry升级了

今天打开Zoundry的时候，发现它已经升级到0.9.137了，赶紧下一个来用。

官方资料里面特别提到了段落操作的改进：

New Enter key behaviour:
Pressing Enter key starts a new paragraph (); Pressing Shift+Enter inserts a line break () within the same paragraph. Use Shift+Enter if you want to start a new line on the same paragraph.

按回车键开始一个新的段落（）；按Shift+回车键是在同一个段落里面换行（）

嘿，两周前我还给他们写过信，说这个事情，他们回信说1、2个月内解决，他们速度还是挺快的，现在这个操作方式方便多了，也更像Dreamweaver了。

另外，Zoundry对自己的不足也作了一下改进：

  • 比如自己不支持复杂HTML（比如table）的编写，就提供了OpenHtmlFile的方法，让用户可以直接打开编辑好的HTML
  • 增加了PasteAsHtml，让用户可以选择直接粘贴HTML内容，或仅粘贴文本内容
  • 为图片加链接的时候不需要鼠标拖拽成高亮状态了，直接在图片上右键点"Create Link"就行了

以上是我比较感兴趣的更新，其他更新请看这里：http://www.zoundry.com/releasenotes.html

不知道Zoundry以后能不能支持多语言包，这样就可以彻底推广了。

今日流水账 - 都是Blog惹的祸

最近关于Blog方面的讨论又多了起来，有些甚至是争论，唉，都是Blog惹的祸。

主要集中在几方面：

  • BSP需要不需要首页？
  • Blogger是精英还是草根？
  • Blog比赛、评比
  • 等等

最近一阵门户网站开始蠢蠢欲动，形容不太恰当，应该是已经运动了（几个月就来一次）~

新浪的Blog2.0出来没多久，搜狐和网易的Blog也跟上来了，新浪的Blog大赛和搜狐的Blog调查更是为这个本来就火的事情再添了一把柴，正在新浪搜狐争第一还次没结束的时候博客网的第二次大赛又来了~

我一直特别羡慕那些写个人类（生活、心情等）Blog的朋友，他们写得很自在，也很有个性……他们也许不会经常更新，也许不在乎自己的Blog是否会上BSP的首页，也许不在乎有多少人会看，也许他们并不在乎自己是不是精英，也许并不在乎什么比赛、评比，也许不在乎Blog有没有Feed，也许不在乎Blog这玩意叫什么中文名字……

这篇流水账又为了Blog而Blog，唉，都是Blog惹的祸~
