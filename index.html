关于邮箱

  • 记得刚上网时（1998年了），填的第一个注册表单就是邮箱，那会儿用的是hotmail和163.net，后来陆陆续续注过20多个邮箱服务，很多后来都没坚持下来，有的是服务停止了，有的是我放弃了。
  • 觉得最好的还是 Gmail，曾几何时也想好好用Gmail，太多无奈，只能在很小的范围内用用。
  • 个人邮箱一直是免费和付费混用，付费的主要是给银行/信用卡使用，免费的就用的比较随意。
  • 办公用的邮件服务起初都是单位自己维护的，谈不上好用还经常被spam，最近几年用腾讯企业邮箱，觉得好用很多。
  • Basecamp 的 HEY 看样子是个不错的服务，以后有条件后可以试试。

自动化AutomationIFTTTAPIwebhook

  • 我现在使用的自动化程度很低，仅限于将部分重复工作变成自动执行，当然交了不少学费，但稳定的部分多次重复使用已经可以提高一点效率了。

    • iOS上的IFTTT、Workflow（快捷指令的前身）

    • 免费版仅提供三个Applet，且只能触发单步网络服务，最快每1小时检查1次

浏览器科技

  • 为什么是互联网企业而不是科技企业？
    • 业务也就是门户、搜索、电商、游戏、社交、广告、云服务等互联网业务为主，哪怕有电动车、手机、物流等实体业务，也多是脱离互联网就玩不转的那种
    • 形象上是挤压衣食住行的利润、炒作宣传、大数据杀熟、误导消费、安全隐患等
    • 也许不该对他们要求太高，就别往科技企业方向努了吧？
    • 但愿我是错的，欢迎他们用实际行动来打脸。
  • 我记得那会儿用 Firefox 代替 IE 是因为 IE 老崩溃，后来用 Chrome 替换 Firefox 是因为 Chrome 更快，现在用 Edge 作为默认浏览器是因为 CPU 占用率比 Chrome 低。
