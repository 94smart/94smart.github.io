除草项目2：升级Wordpress

BlogBlogFeaturedsvnWordpressWordpress随笔

还是把这个Wordpress升级到2.9.2~ 本来想在原来的wordpress 2.7上改进现有Blog的模板，不过发现很多好看的模板都是2.9的，而且好像2.9的插件支持更多一点，所以就升级了。 记忆中还是使用svn更新的方法省事，原来的步骤：

  1. 远程登录服务器（我用命令行ssh）
  2. 进入wordpress目录
  3. 运行svn up
  4. 访问http://xxx.com/wordpress/wp-admin/upgrade.php，点升级
  5. 搞定！

这次想升级的时候发现失效了，最后在wordpress.org查到他们在2.7版时更换了repository，正确的做法是：

  1. 进入wordpress目录
  2. 运行svn switch –relocate http://svn.automattic.com/wordpress/tags/2.7 http://core.svn.wordpress.org/tags/2.7/ 其中2.7是wordpress现有的版本号
  3. 以后再运行svn up就可以获得最新的代码了
  4. 其他都和正常升级是一样的

我还顺便改了现在用的模板，以后不太想放广告了，以后只放友情链接，哦了~ p.s.1.如果svn up不生效，还可以使用svn sw http://core.svn.wordpress.org/tags/2.9.2 blog p.s.2.wordpress功能越来越强，操作也越来越复杂，Blog肯定够了，但好像还是不能取代CMS的存在~

手机版支付宝客户端试用(iPhone,S60)

alipayapp storeiPoneMobileS60V3互联网手机版客户端支付宝淘宝电子商务试用随笔

上周末试用了一下支付宝的手机版客户端，包括1016刚发了一个iPhone版和S60V3版的客户端。 官方博客提供的信息：

支付宝正式推出S60版手机客户端 支付宝手机客户端主要包括如下功能： · (1)通过“我要付款”功能，向其他支付宝用户进行付款； · (2)对淘宝网和其它网站拍下的商品，用支付宝手机客户端付款和确认收货； · (3)账务和交易明细的查询； · (4)S60 和Windows Mobile版本支持用户注册； · (5)建行卡通的开通； · (6)卡通和网汇e充值； · (7)手机话费充值，充值有优惠； · (8)支持公共事业缴费，除了S60 V2等版本以外。公共事业缴费目前支持上海市，其他城市陆续上线中； · (9)支持提现功能； 支付宝手机客户端通过苹果认证 可在App Store免费下载 iPhone版说明： 通过该手机支付宝软件，Iphone用户可以在其手机上实现移动支付功能，包括对个人或淘宝网交易进行付款并确认收货，以及水电煤缴费服务，特别便捷的是该软件还集成了手机话费充值功能，即利用支付宝账户内资金对本手机进行充值。在账户资金流入方面，除通过网银充值外，还可利用该软件在本机直接实现支付宝卡通与网汇e的充值

粗略总结现在支付宝手机客户端的情况：

  1. 针对用户群应该比较高端，使用智能手机且明白支付宝的运作流程。
  2. 主要为方便原支付宝用户使用（交易查询、确认付款等）
  3. 可以间接使用建行支付宝卡通，但安全性有待提高
  4. 不能通过信用卡付费
  5. 暂时没有神州付、手机话费等其他途径为支付宝充值
  6. 还未见到网游充值等常用应用

就现在的情况来看，手机版支付宝客户端现在只是一个试水作品，还不能单独使用，如果有旺旺客户端支持就好了，现有的客户端想顺畅使用，最好还是提前通过PC在支付宝帐户充值。 不过未来阿里应该会把支付宝客户端做得更完善，远比现在的版本要好吧。

装还是不装，这是问题？

PD产品设计产品设计随笔

白鸦和Keso最近的火药味很浓，当然，说的是在产品设计上的事：

鸦说：把自己当作一个平凡的用户，不停的用。 Keso说：别装了，你不是用户 鸦又说：装，是必须的

讨论核心是装还是不装，我觉得都对。 做产品不是只有一种方法，装是一种方法，不装是另一种方法，且还有其他方法。 做产品的目的可能会有不同，但好的、成功的产品有很多共同点，比如可用、易用等。 所以，只要可以将产品做好或者达到产品的既定目标，可以通过各种方法，装还是不装都行吧，殊途同归嘛。 最后，管我叫墙头草好了，我希望达到不装的境界，但更希望装的出神入化。
